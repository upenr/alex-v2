Logging into the App will initiate a sync of global and personal agenda data from the IBMtechU server (ibmtechu.com). This data will be stored locally to your device.

You may want to sync periodically, say once a day or if you have made a change to your agenda on the ibmtechu.com web application.

Internet Connectivety

  • For the purpose of viewing, searching and building a personal agenda plan, internet connectivity is not required.
  • To Submit surveys and check the scheduled status of a session, internet connectivity is required.

Data Synchronization

The Synchronization function pulls down two sets of information from the web application (ibmtechu.com) server.

  • The latest Agenda (All Sessions)
  • Your personal agenda (My Agenda)

When sessions are added to your personal agenda using this app, the sync operation does not sync these back upstream to the web application.

Surveys are automatically sent to the web application and are not part of the sync function.

Personal Agenda Plan (My Agenda, option)

Sessions can be added to your agenda plan using either the web application or this app. Sessions added with ibmtechu.com will appear with a white background vs a lightblue background using this app.

  • Sessions added using ibmtechu.com cannot be removed from your agenda plan using this app. To do this login to ibmtechu.com, remove and re-sync.
  • To add a session to your agenda, select the session from "Agenda Sessions" and select "Add to Agenda".
  • To submit a session survey, select the session in "My Agenda" or "Agenda Session" menu options.
  • When you select a session in "My Agenda", if the device has internet connectivity, the App will report back the status of the session, for example: On Schedule, Cancelled or Moved.

Material Downloads

To download PDF session material, please login to our ibmtechu.com server application.
